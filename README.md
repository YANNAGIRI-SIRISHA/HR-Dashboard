# HR Dashboard – Intern Document Verification

This project is part of the HR Dashboard system used to verify documents submitted by interns.

## Features

- Upload intern documents
- View uploaded documents
- Accept or Reject documents
- Document verification status badges
- Search interns
- Filter by verification status
- Analytics charts for verification progress

## Document Types

- Identity Proof (Aadhar / Voter ID / PAN)
- Address Proof
- College ID Card
- Latest Marksheet
- Internship Offer Letter
- PAN Card
- Passport-size photographs
- Emergency contact details
- Updated Resume

## Analytics

The dashboard includes:

- Verification Progress (Doughnut Chart)
- Verified Documents Status (Bar Chart)
- Documents Verified Per Day (Line Chart)

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
